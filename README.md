# mocker-xl

一款用于提高前端开发效率的可视化工具

![首页](https://images.cnblogs.com/cnblogs_com/xiongxiaolong/1648076/o_2002151241170E5F61EC-3CB7-4E6C-8C0A-27F91DA1A0AC.jpg)

## 功能点

- [x] Mock 数据，支持 `mockjs` 语法
- [x] 支持代理，可以携带 cookie
- [x] 支持延时返回
- [] 支持多状态，失败或成功
- [] 校验参数

## 环境要求

需先安装 `mogodb` 和 `node` 环境

## 技术栈与第三方库

-typescript

Client

- React
- Ant Design, Mock.js, etc...
- Less

Server

- Koa
- MongoDB
- Mongoose
